Binary Puzzle rules

1. Each box should contain either a 1 or a 0.

2. More than 2 equal numbers immediately next to or below each are not allowed.

3. Each row and each column should contain an equal number of zeros and ones.

4. Each row is unique and each column is unique.
Thus, any row/column cannot be exactly equal to another row/column.
